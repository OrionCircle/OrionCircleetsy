Privacy Policy

OrionCircle · last updated 14 July 2026

This policy explains what data the OrionCircle software integration handles when it connects to social platforms — including TikTok — to publish the shop's own promotional content. The integration is a private, single-operator tool used only with accounts the operator personally owns. It has no other users and collects no data from the public.

What we access, and why

When the operator connects a TikTok account, TikTok issues the integration an access token via TikTok Login. Using it, the integration accesses only:

  • Basic account info (user.info.basic) — the account's open ID, display name and avatar, used only to confirm which account is connected and to label uploads correctly.
  • Video upload (video.upload) — permission to send a video file into the connected account's own TikTok Drafts/Inbox, where the operator reviews and posts it.

The integration reads nothing else. It does not access followers, direct messages, analytics, or any other user's content.

What we store

  • The TikTok access and refresh tokens, stored on a private server, readable only by the single system account that uses them (file permissions 600), never exposed publicly.
  • A short list of which of our own videos have already been uploaded, so the same one is not sent twice. This contains no personal data.

We do not store, sell, share, or transfer any TikTok user data to third parties. We do not use it for advertising or profiling. There are no other users whose data could be involved.

How content flows

TikTok fetches each video by pulling it from a temporary public URL on our media server. That URL carries a random unguessable token, serves only that one promotional video, and can be removed after use. No personal data is placed there — only the shop's own marketing video.

Data retention and deletion

Tokens are kept only while the account stays connected. The operator can revoke access at any time from TikTok's own "Manage app permissions" settings, which immediately invalidates the token. On request to the contact below, all stored tokens and upload records are deleted.

Third parties

The shop uses Etsy (sales and delivery), Printify (printing and shipping), and social platforms (TikTok, Instagram, Facebook, YouTube) for promotion. Each has its own privacy policy governing data the operator provides directly to them. This policy covers only the OrionCircle integration.
